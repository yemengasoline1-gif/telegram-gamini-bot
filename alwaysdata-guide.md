# دليل AlwaysData

## الخطوات:
1. سجل في alwaysdata.com
2. اذهب إلى Web → Applications
3. أنشئ Python Application
4. ارفع الملفات عبر FTP
5. عيّن Environment Variables
6. شغل البوت

## معلومات FTP:
- Host: ftp-username.alwaysdata.net
- Username: username
- Password: كلمة مرور حسابك
- Port: 21

## التطبيقات الموصى بها للهاتف:
- AndFTP (لرفع الملفات)
- JuiceSSH (للاتصال بـ SSH)
- GitHub Mobile (لإدارة الكود)
